# AWS S3 + CloudFront Portfolio Hosting

## 📌 Overview
This project demonstrates hosting a static website using AWS S3 and delivering it securely via CloudFront (HTTPS).

## 🚀 Steps Performed
1. Created portfolio website using HTML & CSS
2. Uploaded code to GitHub
3. Created S3 bucket
4. Enabled static website hosting
5. Added bucket policy for public access
6. Uploaded website files
7. Created CloudFront distribution
8. Enabled HTTPS
